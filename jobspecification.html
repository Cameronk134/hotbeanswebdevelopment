Job Specification

Web Developer

A web developer is an IT specialist who develops and takes care of websites and web applications.The responsibilities in this sector include designing, creating and maintaining websites through programming languages such as HTML,CSS and javaScript.

Web Designer

A web designer is someone who works on the creative side when it comes to producing a website. A web designer is responsible for designing visual elements of the website for use online. This is done in order to represent the company's branding and design aesthetic. They create layouts for landing pages, product pages and other areas of the website and test different designs to see which are most effective and which design fits the needs of the request of the website being created.

Web Engineer

A website engineer is someonne who builds websites and applications. Using computer coding skills, they can create web applications and destinations that meet user needs. They can also maintain these applications and monitor them to ensure that their functionality remains intact.
